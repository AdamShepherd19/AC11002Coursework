Choral Shield

My Higher Computing Science project!

This project was undertaken in 2017 for my Higher Computing coursework. It consisted of a Python program that read in data from a file about a series of ticket orders for an event. It then searched through them and identified all tickets for the friday night. These were then written to a file that is then utilised by a database.

The database then imports the data from the new file and arranges it so a query can be run to see how many people are supporting each team and the number of avilable seats in each area.
