Reversi

Java project I am currently working on for University!

This is a project I am currently working on. It is using Java to recreate the classic game Reversi. The program utilisies Object Oriented Programming, 2D arrays and file handling to succesfully implement the game. We worked in groups to come up with the requirements and design for the program and it is now my task to implement it.

Once the code is succesfully implemented, tested and is fully operational, I might venture into creating a graphical interface for the game, to make it more interactive and user-friendly.
